# Hand Writing
본 프로젝트는 Flutter로 개발 되었습니다.

## 사용 방법
![예시 이미지](assets/Hand Writing 예시 이미지.png)

1. 화면에 나타난 문장을 따라 작성 후 해당 이미지 촬영
2. 촬영 된 이미지 확인 후 확인 버튼을 눌러 폰트 선택창으로 이동
3. 생성 된 폰트 중 원하는 폰트를 선택 후 사용하기 누르기
4. 선택된 폰트 사용 및 다운로드
